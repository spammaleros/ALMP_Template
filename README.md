ALMP or Asymmetrical Local Multiplayer Template is a free Blueprint-only template that will help you build games in which VR and non-VR players play on one PC.

**Unreal Engine:** 4.22

**Supported HMDs:**
1. Oculus Rift
2. Oculus Rift S
3. HTC Vive
4. HTC Vive Pro

**Optimizations:**
- Forward shading ON
- Instanced Stereo ON
